# ESP8266_OneNet

基于STM32C8T6最小系统板所所编写的智能家居

/***************接线  ********************/

DHT11         --------->               PB9

BEEP（高电平触发）----->               PA0

  ESP8266
  
    RX        --------->              PA2
    
    TX        --------->              PA3
    
    RST       --------->              PC14
    
/***************************************/

由于容错率大的，也可以直接调整初始化的端口；或者继续增加外设，并将其数据上传到OneNet平台
